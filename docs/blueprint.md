# **App Name**: GST File Automator

## Core Features:

- PDF Upload & Processing: Allows users to upload one or multiple GST invoice PDFs for automatic extraction. Shows upload progress and file names.
- Invoice Type Detection: Automatically detects the type of GST invoice (Service or Spares) based on content analysis.
- Data Extraction & Validation Tool: Extracts key GST data from invoices and validates totals to ensure accuracy.
- Duplicate Invoice Detection: Identifies and flags duplicate invoice uploads using content hashing. Allows users to keep or ignore duplicates.
- Overview Summary: Provides summary cards displaying total invoices, taxable amounts, and GST values.
- Detailed Table Display: Shows a detailed table with one row per invoice and columns matching the Excel output, allowing for visual review.
- Excel Download: Generates a clean, accountant-ready Excel file containing extracted GST data. Download button in the UI.

## Style Guidelines:

- Primary color: Soft blue (#64B5F6) to convey trust and calmness.
- Background color: Light gray (#F5F5F5) for a clean, modern feel.
- Accent color: Teal (#008080) for CTAs and important information. This helps add focus to the areas that require user interaction without being visually overwhelming.
- Body and headline font: 'Inter' for a modern, neutral, readable feel.
- Use simple, professional icons for key actions and information. Consistent style across the app.
- Clear, well-organized layout with distinct sections for uploading, summary, table display, and download.
- Subtle loading animations and transitions to enhance the user experience without being distracting.
